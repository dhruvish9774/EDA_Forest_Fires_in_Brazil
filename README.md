# EDA_Forest_Fires_in_Brazil

In this, we will focus on ‘Brazil’s Amazon Forest Fires Dataset’ and perform some basic analysis using Pandas,Numpy library and visualise data using Matplotlib and Seaborn libraries.

# Attribute information :

# Year : This column contain the year of forest fire.

# State : In this column there are differnce states of Brazil.

# Month : This column contain the month of forest fire.

# Number : This column contain the numbers of forest fire.

# Date : This column contain the date of forest fire
